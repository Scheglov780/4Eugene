Пожалуйста, не размещайте свой дизайн в папке default (old - в ранних версиях).
В этих папках размещены наши темы отображения, которые мы обновляем а Вы сможете использовать
в качестве образца.
Один из рекомендованных путей создания собственной темы - скопировать содержимое
темы default в новую папку и все дальнейшие изменения производить в ней.

 Для того, чтобы переключить используемую тему, нужно изменить параметр
 site_front_theme в админке (или в таблице config) на название папки Вашей темы.
 Например, если папка называется mytheme, значение параметра site_front_theme должно
 быть mytheme.

 Все изменения в папке темы default (old в ранних версиях) будут перезаписываться
 во время обновлений.
